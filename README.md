# Portal Relay Chat

Portal Relay Chat is a simple one way communication page designed for a captive portal style setup. It allows a connected user to enter an identifier and a short message, then send that information back to the hosting device.

## What It Does

This project works like a lightweight message drop box.

A user connects to the portal, opens the page, types in their identifier, writes a message, and submits it. The message is then passed back to the hosting device through the portal request.

This is not a full live chat system. It is designed for simple one way communication from the connected user back to the portal host.

## Main Features

Simple single page HTML design

Black and deep purple cyberpunk theme

Identifier field for the sender name or label

Message field for short relay messages

One way communication

Designed for local captive portal use

Lightweight and easy to upload

## How It Works

1. Upload the HTML file to your captive portal device.

2. Start the portal.

3. A user connects to the WiFi portal.

4. The user opens the portal page.

5. The user enters an identifier and message.

6. The portal sends the submitted information back to the hosting device.
